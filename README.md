## 📖 About
A modern single-page application built with **React** and styled with **CSS/Bootstrap**.  
This project focuses on **professional design**, **smooth animations**, and **theme-based responsiveness** to create a futuristic UI.

---

## 🚀 Features
-  Single Page Layout (No scrolling, clean structure)  
-  Theme colors with transitions & animations  
-  Responsive design with Flexbox  
-  Modular React components  
-  Mobile-friendly UI  
-  Easy to extend  

---

## 🛠️ Tech Stack
- **React.js**
- **JavaScript (ES6+)**
- **CSS3 / Bootstrap 5**
- **Flexbox Layout**
- **Boxicons for icons**
